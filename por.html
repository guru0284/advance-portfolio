Projects

AI-Driven Skill Assessment System for JEE Students

Developed for Studize, a budding EdTech startup, this project focuses on creating a personalized assessment platform for JEE aspirants. The system is designed to evaluate a student’s performance and understanding based on a SOCA model — Strengths, Opportunities, Challenges, and Action Plan. Using AI-based analytics and dynamic feedback mechanisms, the platform identifies specific academic areas where a student excels or struggles and then recommends targeted actions for improvement. The goal is to provide smart learning insights that go beyond traditional test scoring.

Technologies Used: Python, Pandas, NumPy, Scikit-learn, Data Visualization libraries (e.g., Matplotlib, Seaborn), and Flask for backend development.

Tamil Handwritten Character Recognition Using Neural Networks

This project aimed to build a deep learning model capable of recognizing Tamil handwritten characters with high accuracy. Considering the complexity and diversity of Tamil script, the model was trained using convolutional neural networks (CNNs) to handle the intricacies of handwritten strokes and variations. The dataset was preprocessed and normalized to improve model performance and reduce noise. This project showcases the potential of neural networks in regional language processing and character recognition systems.

Technologies Used: Python, TensorFlow, Keras, OpenCV, NumPy.
